PWA / 모바일 홈 화면 아이콘

생성된 파일 (픽셀 크기 고정, manifest와 일치)
  - icon-192.png     … 192×192 (any)
  - icon-512.png     … 512×512 (any)
  - icon-maskable-512.png … 512×512 (maskable, 본문은 캔버스의 약 80% 안)
  - apple-touch-icon.png … 180×180 (iOS 홈 화면)
  - icon.png         … 192×192 (favicon 호환용, icon-192와 동일)

마스터 원본
  - scripts/pwa-icon-source.png (정사각 권장)

아이콘 다시 빌드
  npm run icons

원본을 바꾼 뒤에는 위 명령으로 public/*.png 를 다시 생성하고 배포하세요.
